> 板块：线性代数 · 重要度 ⭐⭐

## 一句话总结

**余弦相似度 = 点积 ÷ 两个向量的长度 = 去掉长度、只看方向的相似度。**

$$\text{cos\_sim}(\vec a,\vec b)=\cos\theta = \frac{\vec a\cdot\vec b}{\lvert\vec a\rvert\,\lvert\vec b\rvert}$$

结果恒在 **[-1, 1]**：`1` 完全同向，`0` 垂直/无关，`-1` 完全反向。

> ⚠️ 输入两个**等长**向量，输出**一个标量**（就是它俩夹角的余弦）。

---

## 1. 为什么点积不够，要发明余弦相似度？

点积（1.5）的几何式是 $\vec a\cdot\vec b=\lvert\vec a\rvert\lvert\vec b\rvert\cos\theta$。
问题在于：它同时被**长度**和**方向**两个因素影响。

- 一个向量方向没变，只是**放大 2 倍**，点积就跟着放大 2 倍。
- 于是「点积大」既可能是**方向真的接近**，也可能只是**某个向量特别长**。

余弦相似度把两个长度都除掉，**只留下 $\cos\theta$**——纯粹的方向关系，不受长度干扰。

| | 点积 $\vec a\cdot\vec b$ | 余弦相似度 $\cos\theta$ |
|---|---|---|
| 受长度影响？ | ✅ 会（长度越大值越大） | ❌ 不会（已归一化） |
| 取值范围 | 任意实数 | 固定 **[-1, 1]** |
| 看的是 | 方向 + 长度 | **只看方向** |

---

## 2. 几何直觉：单位圆上的夹角

把两个向量都缩到**长度为 1**（单位向量），它们就落在单位圆/单位球面上。
这时候长度都 = 1，点积 $=1\cdot1\cdot\cos\theta=\cos\theta$。

**所以：余弦相似度 = 先把向量归一化成单位向量，再做点积。**

$$\cos\theta = \hat a\cdot\hat b,\qquad \hat a=\frac{\vec a}{\lvert\vec a\rvert}$$

**为什么「先归一化再点积」和「先点积再除以长度」是同一回事？**
把归一化写开，用乘除可交换的性质一步步展开就看清了：

$$\hat a\cdot\hat b
= \frac{\vec a}{\lvert\vec a\rvert}\cdot\frac{\vec b}{\lvert\vec b\rvert}
= \frac{\vec a\cdot\vec b}{\lvert\vec a\rvert\,\lvert\vec b\rvert}
= \cos\theta$$

- **先归一化再点积**：$\dfrac{\vec a}{\lvert\vec a\rvert}\cdot\dfrac{\vec b}{\lvert\vec b\rvert}$（每个向量先各自除以自己的长度）
- **先点积再除**：$\dfrac{\vec a\cdot\vec b}{\lvert\vec a\rvert\,\lvert\vec b\rvert}$（先点积，最后一起除）

两者只是把「÷长度」这一步**提前或延后**，乘除顺序可交换，结果完全相同。
> ⚠️ 注意分母是两个长度**相乘**（$\lvert\vec a\rvert\lvert\vec b\rvert$，之积），不是相加。

| 夹角 θ | cos θ | 含义 |
|:---:|:---:|------|
| 0° | **+1** | 完全同向，最相似 |
| 60° | 0.5 | 比较相似 |
| 90° | **0** | 垂直/正交，毫不相关 |
| 120° | -0.5 | 比较相反 |
| 180° | **-1** | 完全反向，最不相似 |

> 💡 记忆锚点：**余弦相似度就是「归一化之后的点积」**。点积懂了，这里只多一步除以长度。

## 3. 手算一遍

$$\vec a = [1, 2, 3],\quad \vec b = [4, 5, 6]$$

**第一步 点积**（1.5 算过）：$\vec a\cdot\vec b = 1\cdot4+2\cdot5+3\cdot6 = 32$

**第二步 两个长度**：

$$\lvert\vec a\rvert=\sqrt{1^2+2^2+3^2}=\sqrt{14}\approx3.742$$
$$\lvert\vec b\rvert=\sqrt{4^2+5^2+6^2}=\sqrt{77}\approx8.775$$

**第三步 相除**：

$$\cos\theta=\frac{32}{\sqrt{14}\,\sqrt{77}}=\frac{32}{\sqrt{1078}}\approx\frac{32}{32.83}\approx0.9746$$

≈ 0.97，非常接近 1 → 两个向量方向很接近。

再看几个 2 维例子感受范围：

| $\vec a$ | $\vec b$ | 点积 | cos 相似度 | 关系 |
|---|---|:---:|:---:|---|
| `[1, 0]` | `[1, 0]` | 1 | **1.0** | 完全同向 |
| `[1, 0]` | `[2, 0]` | 2 | **1.0** | 同向（长度不影响！） |
| `[1, 0]` | `[0, 1]` | 0 | **0.0** | 垂直/无关 |
| `[1, 0]` | `[-1, 0]` | -1 | **-1.0** | 完全反向 |

> 注意第 2 行：点积从 1 变成 2，但**余弦相似度都是 1.0**——这正是我们想要的「只看方向」。

## 4. 和点积、欧氏距离的关系（别搞混）

| 度量 | 公式 | 关注 | 范围 |
|------|------|------|------|
| 点积 | $\vec a\cdot\vec b$ | 方向 + 长度 | 任意实数 |
| **余弦相似度** | $\dfrac{\vec a\cdot\vec b}{\lvert\vec a\rvert\lvert\vec b\rvert}$ | **只看方向** | [-1, 1] |
| 欧氏距离 | $\lvert\vec a-\vec b\rvert$ | 两点直线距离 | [0, +∞) |

- **相似度 vs 距离**：余弦相似度**越大越像**；距离**越小越像**，方向相反。
- **余弦距离** = $1-\cos\theta$，范围 [0, 2]，把「相似度」转成「距离」用（越小越像）。
- **归一化后，三者统一**：若向量都是单位长度，则
  $\lvert\hat a-\hat b\rvert^2 = 2-2\,\hat a\cdot\hat b = 2(1-\cos\theta)$——
  余弦相似度越大 ⇔ 欧氏距离越小。所以很多向量库归一化后直接用点积当余弦相似度。

## 5. 两个高频坑

### 5.1 零向量：分母为 0

如果某个向量是全 0，$\lvert\vec 0\rvert=0$，公式会**除以 0**（NumPy 给 `nan` 或警告）。
实践中要么跳过零向量，要么给分母加一个极小值 `eps`：

```python
cos = a @ b / (np.linalg.norm(a) * np.linalg.norm(b) + 1e-8)
```

### 5.2 别忘了「先归一化再批量点积」的等价写法

对一堆向量算两两余弦相似度时，**先各自归一化成单位向量，再做矩阵乘法**，
一次 `A_norm @ B_norm.T` 就是所有 pair 的余弦相似度矩阵（比逐对除法快得多）。

## 6. 和 LLM 的联系（为什么⭐⭐）

### 6.1 embedding 语义相似度

把两句话各编码成一个向量（embedding），**余弦相似度就是它们的「语义有多像」**：

- ≈ 1：意思几乎一样（"猫" vs "小猫"）
- ≈ 0：不相关（"猫" vs "银行利率"）
- < 0：语义相反 / 冲突

### 6.2 向量检索 / RAG 召回

RAG 的召回步骤：把 query 和所有文档 chunk 都编码成向量，
**按余弦相似度排序，取 Top-K 最相关的塞进上下文**。这就是向量数据库（FAISS、Milvus 等）的核心操作。

> 为什么用余弦而不是原始点积？因为文本长短会让 embedding 长度不同，
> 余弦相似度**去掉长度只比方向**，不会因为「文档长」就虚高分数，更公平。

### 6.3 和注意力的分工

- 注意力内部用的是 **scaled dot-product**（点积，1.5），因为 Q/K 已在同一空间、要保留幅度信息。
- 检索/RAG 这类「跨文本比语义」的场景用**余弦相似度**，因为要抹掉长度只看方向。
- 一句话：**注意力比点积，检索比余弦。**
