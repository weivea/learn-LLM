## 1 · 从期望计算方差（20 分）

继续使用 3.7 的非均匀骰子：

| 点数 $x$ | 1 | 2 | 3 | 4 | 5 | 6 |
|---:|---:|---:|---:|---:|---:|---:|
| $P(X=x)$ | 0.05 | 0.10 | 0.15 | 0.20 | 0.20 | 0.30 |

已知：

$$
\mathbb E[X]=4.3,
\qquad
\mathbb E[X^2]=20.8
$$

1. 使用 $\mathbb E[X^2]-(\mathbb E[X])^2$ 计算方差。
2. 写出使用平方偏差直接计算方差的完整求和式。
3. 计算标准差，保留三位小数。
4. 方差和标准差分别使用什么单位？
5. 用一句话解释方差与标准差各自表达什么。

✍️ 你的答案：
1. $\operatorname{Ver}(X) = \mathbb E[X^2]-(\mathbb E[X])^2=20.8-4.3^2=2.31$

2.
$$
\begin{aligned}
\operatorname{Var}(X)
&=\sum_i (x_i-\mu)^2p_i\\
&=\sum_i (x_i-\mathbb E[X])^2p_i\\
&=(1-4.3)^2(0.05)+(2-4.3)^2(0.10)\\
&\quad +(3-4.3)^2(0.15)+(4-4.3)^2(0.20)\\
&\quad +(5-4.3)^2(0.20)+(6-4.3)^2(0.30)\\
&=2.31
\end{aligned}
$$

3.
$$
\begin{aligned}
\operatorname{Std}(X)
&=
\sqrt{2.31}\\
&\approx
1.520
\end{aligned}
$$
4. 方差是原数据单位的平方；标准差与原数据单位相同

5. 方差描述它围绕中心波动多大，标准差则把这种波动还原到与原数据相同的单位。

## 3 · 总体方差、样本方差与 `ddof`（20 分）

观察到数据：

```text
[2, 4, 6]
```

1. 计算均值。
2. 把这三个数视为完整总体，计算总体方差与总体标准差。
3. 把这三个数视为总体的一个样本，计算常用的样本方差与样本标准差。
4. 为什么样本方差通常除以 $n-1$？
5. 补全并写出四个结果：

   ```python
   import numpy as np

   x = np.array([2.0, 4.0, 6.0])

   population_variance = ...
   sample_variance = ...
   population_std = ...
   sample_std = ...
   ```

✍️ 你的答案：
1. 均值= 4
2. $\text{总体方差}=((2.0-4)^2 + (4.0-4)^2+(6.0-4)^2)/3 = 2.667$ \
   $\text{总体标准差}=\sqrt{\text{总体方差}}=1.633$
3. $\text{总体方差}=((2.0-4)^2 + (4.0-4)^2+(6.0-4)^2)/2 = 4$ \
   $\text{总体标准差}=\sqrt{\text{总体方差}}=2$
4. 因为使用样本均值后，所有偏差之和必须为 $0$，所以只有 $n-1$ 个偏差可以自由变化，即只有 $n-1$ 个自由度。若仍除以 $n$，会系统性低估未知的总体方差；除以 $n-1$ 可以修正这种低估，使样本方差成为总体方差的无偏估计。

5.

```python
import numpy as np

x = np.array([2.0, 4.0, 6.0])

population_variance = np.var(x, ddof=0)  # 2.6666666666666665
sample_variance = np.var(x, ddof=1)      # 4.0
population_std = np.std(x, ddof=0)       # 1.632993161855452
sample_std = np.std(x, ddof=1)           # 2.0
```

## 4 · 手算 LayerNorm（20 分）

某个 token 的隐藏向量是：

$$
\boldsymbol h=[1,2,3,4]
$$

暂时忽略 $\varepsilon$，并令 $\gamma=1,\beta=0$。

1. 计算该向量的均值。
2. 使用分母 $d=4$ 计算方差和标准差。
3. 计算标准化后的四个值，保留四位小数。
4. 验证标准化结果的均值为 $0$、方差为 $1$。
5. 若隐藏状态 shape 是 `(batch, seq_len, d_model)`，标准 LayerNorm 通常沿哪一维计算？实际公式为什么要加入 $\varepsilon$？

✍️ 你的答案：
1. 均值=2.5
2. $Ver(h)_{d-4} =((1-2.5)^2+(2-2.5)^2+(3-2.5)^2+(4-2.5)^2)/4=1.25$ \
   $Std(j) = \sqrt {1.25}=1.118$
3. 根据标准化公式：

$$
\operatorname{LayerNorm}(h_j)
=
\gamma_j
\frac{h_j-\mu}{\sqrt{\sigma^2+\varepsilon}}
+\beta_j
$$

因为忽略 $\varepsilon$，且 $\gamma=1,\beta=0$，所以：

$$
\begin{aligned}
\operatorname{LayerNorm}(\boldsymbol h)
&=
\frac{\boldsymbol h-2.5}{\sqrt{1.25}}\\
&=
\left[
\frac{-1.5}{\sqrt{1.25}},
\frac{-0.5}{\sqrt{1.25}},
\frac{0.5}{\sqrt{1.25}},
\frac{1.5}{\sqrt{1.25}}
\right]\\
&\approx
[-1.3416,-0.4472,0.4472,1.3416]
\end{aligned}
$$

**4.** 标准化结果的均值为：

$$
\frac{-1.3416-0.4472+0.4472+1.3416}{4}=0
$$

使用未四舍五入的精确值，其方差为：

$$
\frac{
\left(-\frac{3}{\sqrt{5}}\right)^2+
\left(-\frac{1}{\sqrt{5}}\right)^2+
\left(\frac{1}{\sqrt{5}}\right)^2+
\left(\frac{3}{\sqrt{5}}\right)^2
}{4}
=
\frac{\frac95+\frac15+\frac15+\frac95}{4}
=1
$$

因此，标准化结果的均值为 $0$、方差为 $1$。

**5.** 标准 LayerNorm 通常沿最后一维 `d_model` 计算，即对每个 batch 中每个 token 的隐藏向量分别进行标准化。公式中加入 $\varepsilon$，是为了避免方差为 $0$ 或非常小时出现除零，并提高浮点运算的数值稳定性。

### 作答 C · NumPy

**5.** 实现如下：

   ```python
   import numpy as np


   def weighted_variance(values, probabilities):
       outcomes = np.asarray(values, dtype=np.float64)
       weights = np.asarray(probabilities, dtype=np.float64)

       if outcomes.ndim != 1 or outcomes.size == 0:
           raise ValueError("values 必须是一维非空数组")
       if weights.ndim != 1 or weights.size == 0:
           raise ValueError("probabilities 必须是一维非空数组")
       if outcomes.shape != weights.shape:
           raise ValueError("values 与 probabilities 形状必须相同")
       if not np.all(np.isfinite(outcomes)):
           raise ValueError("values 不能包含 NaN 或 Inf")
       if not np.all(np.isfinite(weights)):
           raise ValueError("probabilities 不能包含 NaN 或 Inf")
       if np.any(weights < 0.0):
           raise ValueError("概率不能为负数")
       if not np.isclose(weights.sum(), 1.0):
           raise ValueError("概率之和必须为 1")

       mean = np.sum(outcomes * weights)
       return float(np.sum((outcomes - mean) ** 2 * weights))


   def standardize_last_dim(values, eps=1e-5):
       data = np.asarray(values, dtype=np.float64)

       if data.ndim == 0 or data.size == 0:
           raise ValueError("values 至少需要一维非空输入")
       if not np.all(np.isfinite(data)):
           raise ValueError("values 不能包含 NaN 或 Inf")
       if not np.isfinite(eps) or eps < 0.0:
           raise ValueError("eps 必须是非负有限数")

       mean = data.mean(axis=-1, keepdims=True)
       centered = data - mean
       variance = np.mean(centered**2, axis=-1, keepdims=True)
       denominator = np.sqrt(variance + eps)

       if np.any(denominator == 0.0):
           raise ValueError("方差为 0 时 eps 必须大于 0")

       return centered / denominator
   ```

## 第 1 题

使用等价公式：

$$
\begin{aligned}
\operatorname{Var}(X)
&=
\mathbb E[X^2]-(\mathbb E[X])^2\\
&=
20.8-4.3^2\\
&=
20.8-18.49\\
&=
2.31
\end{aligned}
$$

直接使用平方偏差：

$$
\begin{aligned}
\operatorname{Var}(X)
&=
(1-4.3)^2(0.05)
+(2-4.3)^2(0.10)\\
&\quad
+(3-4.3)^2(0.15)
+(4-4.3)^2(0.20)\\
&\quad
+(5-4.3)^2(0.20)
+(6-4.3)^2(0.30)\\
&=
2.31
\end{aligned}
$$

标准差：

$$
\operatorname{Std}(X)
=
\sqrt{2.31}
\approx
1.520
$$

若原变量单位是“点数”，方差单位是“点数的平方”，标准差单位仍是“点数”。方差表示平方偏差的平均大小，适合推导；标准差表示与原变量同单位的波动尺度，更容易解释。

## 第 2 题

期望使用线性性质：

$$
\begin{aligned}
\mathbb E[Y]
&=
\mathbb E[-2X+5]\\
&=
-2\mathbb E[X]+5\\
&=
-15
\end{aligned}
$$

方差与标准差：

$$
\begin{aligned}
\operatorname{Var}(Y)
&=
(-2)^2\operatorname{Var}(X)\\
&=
4(9)\\
&=
36
\end{aligned}
$$

$$
\operatorname{Std}(Y)
=
|-2|\operatorname{Std}(X)
=
2(3)
=
6
$$

对标准化变量：

$$
\mathbb E[Z]
=
\frac{\mathbb E[X]-10}{3}
=
0
$$

$$
\operatorname{Var}(Z)
=
\frac{1}{3^2}\operatorname{Var}(X)
=
1
$$

加常数会让每个值和均值一起平移相同距离，所以各值与均值的偏差不变。正确缩放规律是：

$$
\operatorname{Var}(aX+b)
=
a^2\operatorname{Var}(X)
$$

不是乘 $a$。

```python
import numpy as np


def weighted_variance(values, probabilities):
    outcomes = np.asarray(values, dtype=np.float64)
    weights = np.asarray(probabilities, dtype=np.float64)

    if outcomes.ndim != 1 or outcomes.size == 0:
        raise ValueError("values 必须是一维非空数组")
    if weights.ndim != 1 or weights.size == 0:
        raise ValueError("probabilities 必须是一维非空数组")
    if outcomes.shape != weights.shape:
        raise ValueError("values 与 probabilities 形状必须相同")
    if not np.all(np.isfinite(outcomes)):
        raise ValueError("values 不能包含 NaN 或 Inf")
    if not np.all(np.isfinite(weights)):
        raise ValueError("probabilities 不能包含 NaN 或 Inf")
    if np.any(weights < 0.0):
        raise ValueError("概率不能为负数")
    if not np.isclose(weights.sum(), 1.0):
        raise ValueError("概率之和必须为 1")

    mean = float(outcomes @ weights)
    return float(((outcomes - mean) ** 2) @ weights)


def standardize_last_dim(values, eps=1e-5):
    data = np.asarray(values, dtype=np.float64)

    if data.ndim == 0 or data.size == 0:
        raise ValueError("values 至少需要一维非空输入")
    if not np.all(np.isfinite(data)):
        raise ValueError("values 不能包含 NaN 或 Inf")
    if not np.isfinite(eps) or eps < 0.0:
        raise ValueError("eps 必须是非负有限数")

    mean = data.mean(axis=-1, keepdims=True)
    centered = data - mean
    variance = np.mean(
        centered**2,
        axis=-1,
        keepdims=True,
    )
    denominator = np.sqrt(variance + eps)
    if np.any(denominator == 0.0):
        raise ValueError("方差为 0 时 eps 必须大于 0")
    return centered / denominator
```
