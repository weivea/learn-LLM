## 1 · 医学检测与自然频数（20 分）

## 3 · 连续证据更新（20 分）

盒中随机选择了一枚硬币，初始时两种假设概率相同：

```text
H_fair：公平硬币，P(正面 | H_fair) = 0.5
H_bias：偏正面硬币，P(正面 | H_bias) = 0.8
```

1. 第一次观察到正面后，计算 $P(H_{\text{bias}}\mid\text{正面})$。
2. 连续观察到两个正面后，计算 $P(H_{\text{bias}}\mid\text{正面,正面})$。
3. 第三次观察到反面后，计算 $P(H_{\text{bias}}\mid\text{正面,正面,反面})$。
4. 为什么第三次的反面会降低对偏正面硬币的相信程度？
5. 用一句话解释“后验成为下一轮先验”。

✍️ 你的答案：
1. 未归一化权重：
    $$
    \begin{aligned}
    是公平硬币且出现正面的概率：
    w_{\text{fair}}&=0.5\times0.5=0.25\\
    是偏正面硬币且出现正面的概率：
    w_{\text{bias}}&=0.5\times0.8=0.40
    \end{aligned}
    $$
    所以：
    $$
    P(H_{\text{bias}}\mid\text{正面})
    =
    \frac{0.40}{0.25+0.40}
    \approx
    0.6154
    $$

2. 未归一化权重：
    $$
    \begin{aligned}
    是公平硬币且出现两次正面的概率：
    w_{\text{fair}}&=0.5\times0.5^2=0.125\\
    是偏正面硬币且出现两次正面的概率：
    w_{\text{bias}}&=0.5\times0.8^2=0.32
    \end{aligned}
    $$

    所以：

    $$
    P(H_{\text{bias}}\mid\text{正面,正面})
    =
    \frac{0.32}{0.125+0.32}
    \approx
    0.7191
    $$
3. 未归一化权重：
    $$
    \begin{aligned}
    是公平硬币且出现两次正面一次反面的概率：
    w_{\text{fair}}&=0.5\times0.5^2\times 0.5=0.0625\\
    是偏正面硬币且出现两次正面一次反面的概率：
    w_{\text{bias}}&=0.5\times0.8^2\times 0.2=0.064
    \end{aligned}
    $$

    所以：

    $$
    P(H_{\text{bias}}\mid\text{正面,正面})
    =
    \frac{0.064}{0.0625+0.064}
    \approx
    0.5059
    $$
4. 由于$P(正面 | H_bias) > P(正面 | -H_bias)$, 但第三次没能出现正面，故而偏正面硬币的可行度记忆降低了，\
   换一个角度  $P(反面面 | H_fair) = 0.5 > P(反面 | -H_bias)=0.2$, 第三次出现反面它增加了是公平硬币的可行度，那便降低了是偏正面硬币的可行度
5. 吸收当前证据得到的后验概率，会作为观察下一条新证据时的先验概率。

## 4 · MLE、MAP 与 LLM 边界（20 分）

有两个互斥且穷尽的候选模型（即真实模型被假定为二者之一）：

| 模型 | 先验 $P(M)$ | 数据似然 $P(D\mid M)$ |
|---|---:|---:|
| 简单模型 | 0.90 | 0.30 |
| 复杂模型 | 0.10 | 0.60 |

1. MLE 会选择哪个模型？为什么？
2. 计算两个模型的后验，并判断 MAP 会选择哪个模型。
3. 为什么 MLE 与 MAP 可能得到不同结果？
4. LLM 的 $p_\theta(x_t\mid x_{<t})$ 是条件概率。为什么不能仅凭这一点就说“标准 LLM 推理一定在显式执行贝叶斯参数更新”？
5. 为什么 LLM 给某答案较高 token 概率，不等于该答案为真的概率较高？

✍️ 你的答案：
1. MLE叫“最大似然估计”，自然是选似然较大的“复杂模型”
2. 后验：[0.8182, 0.1818], MAP会选择简单模型
3. 因为MLE比较似然，MAP比较后验
4. $p_\theta(x_t\mid x_{<t})$ 表示在参数 $\theta$ 和前文 $x_{<t}$ 给定时，下一个 token 的条件概率分布。标准 LLM 推理时，参数 $\theta$ 通常保持不变，只是前文改变了模型内部的激活和输出分布；而显式贝叶斯参数更新需要根据证据计算并更新参数的后验 $P(\theta\mid D)$。因此，使用条件概率不等于在执行贝叶斯参数更新。
5. 较高的 token 概率只表示模型更倾向于生成该答案，即它更像训练数据和当前上下文中的合理续写，并不表示它更符合客观事实。模型可能受到错误或过时的训练数据、偏差、提示方式以及知识缺失的影响，所以生成概率不是答案为真的概率。

## 5 · NumPy 实现（20 分）

补全通用贝叶斯更新函数：

```python
import numpy as np


def bayes_update(priors, likelihoods):
    pass
```

要求：

- `priors` 与 `likelihoods` 都必须是一维非空有限数组
- 二者形状必须相同
- `priors` 非负且概率和为 $1$
- 每个 likelihood 满足 $0\le p\le1$
- 证据概率必须大于 $0$
- 返回 `(posterior, evidence)`

使用以下断言验证：

```python
posterior, evidence = bayes_update(
    np.array([0.01, 0.99]),
    np.array([0.99, 0.05]),
)
assert np.isclose(evidence, 0.0594)
assert np.allclose(posterior, [1 / 6, 5 / 6])
assert np.isclose(posterior.sum(), 1.0)

posterior, evidence = bayes_update(
    np.array([0.7, 0.3]),
    np.array([0.2, 0.8]),
)
assert np.isclose(evidence, 0.38)
assert np.allclose(posterior, [0.14 / 0.38, 0.24 / 0.38])
```

✍️ 你的答案：

```python
import numpy as np


def bayes_update(priors, likelihoods):
    priors = np.asarray(priors, dtype=np.float64)
    likelihoods = np.asarray(likelihoods, dtype=np.float64)

    if priors.ndim != 1 or priors.size == 0:
        raise ValueError("priors 必须是一维非空数组")
    if likelihoods.ndim != 1 or likelihoods.size == 0:
        raise ValueError("likelihoods 必须是一维非空数组")
    if priors.shape != likelihoods.shape:
        raise ValueError("priors 与 likelihoods 的形状必须相同")

    if not np.all(np.isfinite(priors)):
        raise ValueError("priors 必须全部为有限数值")
    if not np.all(np.isfinite(likelihoods)):
        raise ValueError("likelihoods 必须全部为有限数值")

    if np.any(priors < 0):
        raise ValueError("priors 不能包含负数")
    if not np.isclose(priors.sum(), 1.0):
        raise ValueError("priors 的概率和必须为 1")

    if np.any((likelihoods < 0) | (likelihoods > 1)):
        raise ValueError("每个 likelihood 必须满足 0 <= p <= 1")

    weighted = priors * likelihoods
    evidence = float(weighted.sum())
    if evidence <= 0:
        raise ValueError("证据概率必须大于 0")

    posterior = weighted / evidence
    return posterior, evidence
```

核心计算：先用 `priors * likelihoods` 得到未归一化后验权重，再求和得到证据概率，最后除以证据概率完成归一化。

## 第 1 题

在 $10000$ 人中：

```text
患病人数：
    10000 × 0.01 = 100

真阳性人数：
    100 × 0.99 = 99

未患病人数：
    10000 - 100 = 9900

假阳性人数：
    9900 × 0.05 = 495

总阳性人数：
    99 + 495 = 594
```

所以：

$$
P(\text{患病}\mid\text{阳性})
=
\frac{99}{594}
=
\frac16
\approx
16.67\%
$$

答案不是 $99\%$，因为 $99\%$ 是：

$$
P(\text{阳性}\mid\text{患病})
$$

题目求的是反方向：

$$
P(\text{患病}\mid\text{阳性})
$$

疾病先验只有 $1\%$，健康人基数很大，所以 $5\%$ 的假阳性率会产生 $495$ 个假阳性，远多于 $99$ 个真阳性。

四个角色：

```text
先验：
    P(患病) = 0.01

似然：
    P(阳性 | 患病) = 0.99
    计算完整分母时还需要 P(阳性 | 未患病) = 0.05

证据：
    P(阳性)
    = 0.99 × 0.01 + 0.05 × 0.99
    = 0.0594

后验：
    P(患病 | 阳性)
    = 1/6
```

## 第 2 题

未归一化权重：

$$
\begin{aligned}
w_1
&=
P(H_1)P(E\mid H_1)\\
&=
0.7\times0.2\\
&=
0.14
\end{aligned}
$$

$$
\begin{aligned}
w_2
&=
P(H_2)P(E\mid H_2)\\
&=
0.3\times0.8\\
&=
0.24
\end{aligned}
$$

证据概率：

$$
P(E)=0.14+0.24=0.38
$$

后验：

$$
P(H_1\mid E)
=
\frac{0.14}{0.38}
\approx
0.3684
$$

$$
P(H_2\mid E)
=
\frac{0.24}{0.38}
\approx
0.6316
$$

证据更支持 $H_2$，因为：

$$
P(E\mid H_2)=0.8
>
P(E\mid H_1)=0.2
$$

虽然 $H_2$ 的先验较低，但新证据在 $H_2$ 为真时更容易出现。似然优势足够大时，可以克服先验劣势。

$P(H_2\mid E)$ 不等于 $P(E\mid H_2)$：

- $P(E\mid H_2)$：假设 $H_2$ 为真时，证据出现的概率
- $P(H_2\mid E)$：证据已经出现后，假设 $H_2$ 为真的概率

## 第 3 题

初始先验：

$$
P(H_{\text{fair}})
=
P(H_{\text{bias}})
=
0.5
$$

观察一次正面后：

$$
\begin{aligned}
P(H_{\text{bias}}\mid\text{正面})
&=
\frac{0.5\times0.8}
{0.5\times0.5+0.5\times0.8}\\
&=
\frac{0.40}{0.65}\\
&\approx
0.6154
\end{aligned}
$$

连续两个正面后：

$$
\begin{aligned}
P(H_{\text{bias}}\mid\text{正面,正面})
&=
\frac{0.5\times0.8^2}
{0.5\times0.5^2+0.5\times0.8^2}\\
&=
\frac{0.32}{0.445}\\
&\approx
0.7191
\end{aligned}
$$

再观察一个反面：

$$
\begin{aligned}
P(H_{\text{bias}}\mid\text{正面,正面,反面})

\frac{0.5\times0.8^2\times0.2}
{0.5\times0.5^2\times0.5

&\approx
0.5059

## 第 4 题

MLE 只比较数据似然：

```text
简单模型：0.30
复杂模型：0.60
```

所以 MLE 选择复杂模型。

未归一化后验权重：

$$
\begin{aligned}
w_{\text{简单}}&=0.90\times0.30=0.27\\
w_{\text{复杂}}&=0.10\times0.60=0.06
\end{aligned}
$$

归一化：

$$
P(\text{简单模型}\mid D)
=
\frac{0.27}{0.33}
\approx
0.8182
$$

$$
P(\text{复杂模型}\mid D)
=
\frac{0.06}{0.33}
\approx
0.1818
$$

所以 MAP 选择简单模型。

MLE 只比较 $P(D\mid M)$，MAP 比较：

$$
P(D\mid M)P(M)
$$

先验不均匀时，二者可能选择不同模型。

$p_\theta(x_t\mid x_{<t})$ 是给定前文时的下一个 token 条件分布，但标准 LLM 推理时参数 $\theta$ 通常固定。上下文会改变激活和输出分布，却不代表模型显式计算并更新参数后验 $P(\theta\mid D)$。

token 概率反映模型根据训练数据和上下文形成的输出倾向。模型可能受错误数据、偏差、提示方式或知识缺失影响，因此高 token 概率不构成事实正确性的保证。

```python
import numpy as np


def bayes_update(priors, likelihoods):
    prior_values = np.asarray(priors, dtype=np.float64)
    likelihood_values = np.asarray(
        likelihoods,
        dtype=np.float64,
    )

    if prior_values.ndim != 1 or prior_values.size == 0:
        raise ValueError("priors 必须是一维非空数组")
    if (
        likelihood_values.ndim != 1
        or likelihood_values.size == 0
    ):
        raise ValueError(
            "likelihoods 必须是一维非空数组"
        )
    if prior_values.shape != likelihood_values.shape:
        raise ValueError(
            "priors 与 likelihoods 的形状必须相同"
        )
    if not np.all(np.isfinite(prior_values)):
        raise ValueError("priors 不能包含 NaN 或 Inf")
    if not np.all(np.isfinite(likelihood_values)):
        raise ValueError(
            "likelihoods 不能包含 NaN 或 Inf"
        )
    if np.any(prior_values < 0.0):
        raise ValueError("priors 不能包含负数")
    if not np.isclose(prior_values.sum(), 1.0):
        raise ValueError("priors 的概率之和必须为 1")
    if np.any(likelihood_values < 0.0) or np.any(
        likelihood_values > 1.0
    ):
        raise ValueError(
            "每个 likelihood 必须满足 0 <= p <= 1"
        )

    weighted = prior_values * likelihood_values
    evidence = float(weighted.sum())
    if evidence <= 0.0:
        raise ValueError("证据概率必须大于 0")

    posterior = weighted / evidence
    return posterior, evidence
```
