# 数学知识点掌握度追踪表

> 配套文档：[大模型学习路线.md](大模型学习路线.md) 阶段 0
> 用法：逐个知识点学习，学完在「掌握程度」列自评，定期回顾标 ⚠️ 的薄弱项。
> 目标不是数学系的严谨，而是**看懂公式 + 能对应到代码/张量运算**。

## 📊 图例说明

**重要度**（对学 LLM 的重要性）
- ⭐⭐⭐ 核心，必须吃透，后面天天用
- ⭐⭐ 重要，需要理解
- ⭐ 了解即可，混个脸熟

**掌握程度**（自评，学完后填）
- `⬜ 未开始`
- `🟡 学过但模糊`（看得懂别人推，自己写不出来）
- `🟢 能独立复现`（能不看资料写公式/代码）
- `✅ 能讲给别人听`（能解释原理和直觉）
- `⚠️ 需重点复习`（标记薄弱项，定期回来看）

**学习资源**：每行「学习资源」列给出该知识点最推荐的一个入口，点开即学。常用来源：
- 🎬 **3B1B** = 3Blue1Brown（直觉最好，先看它）｜[线代](https://www.youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab)·[微积分](https://www.youtube.com/playlist?list=PLZHQObOWTQDMsr9K-rj53DwVRMYO3t5Yr)·[神经网络](https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi)
